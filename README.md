# Flask Authentication API

This project consists in create a Authentication API using [Flask](https://flask.palletsprojects.com/en/3.0.x/).

## Requirements

- The API must to have the following endpoints:
  - Login User;
  - Logout User;
  - Create User;
  - Get User;
  - Update User;
  - Delete User;
